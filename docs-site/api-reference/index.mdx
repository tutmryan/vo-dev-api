# Introduction

## GraphQL API schema

This section provides comprehensive documentation for the VO GraphQL API schema. The API is the core of the platform, enabling rich and varied integration options for building compelling consumer applications and administrative user experiences.

## Getting started with the API

The VO API follows GraphQL conventions, providing a flexible and powerful way to query and manipulate data. If you're new to GraphQL, we recommend reviewing the [official GraphQL documentation](https://graphql.org/learn/) before diving into our API.
