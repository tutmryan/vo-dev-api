An update that introduces MFA support in Entra ID and the ability for a credential holder to manage their credential operations using the Concierge.

<!-- truncate -->

## 🚀 What’s new
### New features
- Suspend and resume partners
- Delete unused issuees

### Enhancements
- Verbose EAM authentication logging option
- Apply identity ID constraint when supplying identity input to presentation request
- Copy button for partner DID, issuer and tenant values
- Show presentation receipt in presentation demo
- Add copy to clipboard button for long lines in presentation demo

## 🛠️ Notable fixes
- Potential for OIDC key rotation error at startup
- Prevent special `VerifiableCredential` match-all type from being used as a credential type
- Fix bug with optional claims generating validation errors on issuance (errors will disappear after re-saving contract)
- RBAC for remote issuance was incorrectly applied to issuance route
- Issuance builder no longer needs to enforce selection of issuee for partner presentations
- Fix DID column font size in partner table
- Use consistent navigation after editing partner (return to details page, not list page)
- Validate against use of claim named `photo` which is used for face check images
