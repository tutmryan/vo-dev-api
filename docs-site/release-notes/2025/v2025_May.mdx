## v2025.5.0

### 🚀 What's new

#### New features

- Composer now allows the exporting and importing of Credentials and Templates
- Composer now allows administrators to make changes to the instance configuration
  - Add or remove additional Entra tenants
  - Add or remove issuer labels
  - Add or remove application labels
  - Adjust CORS Origin headers
- OIDC control parameter `vc_constraint_value` can now be set to dynamically resolve to the `login_hint`

#### Enhancements

- Support setting a custom log level in the API service
- The API generates an error when trying to create a template that is nested more than four levels deep
- Composer disables templates from being selected as a parent if they cannot be nested further

### 🛠️ Notable fixes

- Fix OIDC error rendering would output text
