December 2025 releases include OIDC provider enhancements with JAR support, improved security measures, and new credential management features.

<!-- truncate -->

## v2025.12.2

### 🚀 What's new

#### New features

- Added support for JWT-secured authorization requests (JAR) for enhanced security in OpenID Connect flows

#### Enhancements

- Added ability to use local version of VO SDK for development
- Enhanced logo spacing and improved horizontal layout breakpoint for better visual presentation
- Renamed Apollo Sandbox to Apollo Studio in developer tools

### 🛠️ Notable fixes

- Fixed checkbox selection issue on remote issuance page in Composer

---

## v2025.12.1

### 🚀 What's new

#### New features

- Added contract public manifest URL endpoint for external integrations

#### Enhancements

- Adopted semantic versioning and released version 1.0.0
- Enhanced security by locking down limited access tokens used for mdoc presentations
- Improved messaging during presentation request final stage
- Enhanced CORS test success and error messaging in Composer

### 🛠️ Notable fixes

- Fixed issue where permitted partners weren't persisting when updating OIDC clients

---

## v2025.12.0

### 🚀 What's new

#### New features

- Added credential revoker functionality in Composer
- Introduced contract public manifest URL endpoint

#### Enhancements

- Locked down JavaScript client version to prevent potential vulnerabilities

### 🛠️ Notable fixes

- Fixed security vulnerability in GraphQL mutation for authentication presentation request in OIDC flow
