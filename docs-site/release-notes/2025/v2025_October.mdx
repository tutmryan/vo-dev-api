## v2025.10.0

### 🚀 What's new

#### New features

- Introduced Identity Stores feature for managing and querying multiple identity data sources with enhanced access controls and improved UI

#### Enhancements

- Added one-click navigation to return to previously completed steps with stable positioning of the Next button for improved workflow
- Enhanced photo capture widget with MediaPipe-based face detection for improved accuracy and better sizing
- Added visual indication when secret values have been previously entered in configuration fields
- Improved Identity Store UI with better user experience and expanded access permissions

### 🛠️ Notable fixes

- Fixed duplicate identity entries when ordering by name
- Resolved Chromium browser issues with back/forward cache (BFCache) functionality
