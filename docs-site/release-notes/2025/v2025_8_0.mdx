Added management of enterprise issuee stores and identity mapping, instance configuration via the Composer UI, completed a branding refresh and fixed critical issues for EAM MFA verification.

<!-- truncate -->

## 🚀 What’s new
### New features
- Introduced management of issuee stores to improve enterprise identity mapping
- Instance configuration is available via the Composer

### Enhancements
- Set the app label for the 'Authentication Gateway' via system config
- Updated branding throughout the platform to reflect the new logo

## 🛠️ Notable fixes
- Resolved an issue in support of Microsoft External Authentication Methods (EAM) where verification failed when a `knowledge` factor was requested
- Addressed a problem where the photo capture UI never became 'ready' in Safari on iPhone
