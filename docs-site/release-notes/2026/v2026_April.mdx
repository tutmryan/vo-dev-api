April 2026 releases introduce a unified credentials view in Composer, identity proofing support with a restructured OIDC client configuration, tighter Temporary Access Pass eligibility controls in Concierge, access control improvements for remote issuance, enhanced presentation flow usability, a visual indicator for credential type access, and fixes for in-person offer expiry and photo capture.

<!-- truncate -->

## v2026.4.1

### 🚀 What's new

#### New features

- **Unified credentials view in Composer** - A new Credentials section provides a consolidated view of credential records across all issuance methods, with filtering, sorting and load-more pagination. A detailed view shows offer details, issuance history, presentations and communications, and supports cancelling pending offers and resending notifications.
- **Identity proofing for OIDC clients** - OIDC client configuration in Composer now supports identity proofing, with a restructured client form that makes setup clearer.

#### Enhancements

- **Client-supplied OIDC authentication parameters and response types** - The API now accepts additional client-supplied authentication parameters and response types during OIDC flows, giving relying parties more flexibility.
- **Stricter Temporary Access Pass eligibility in Concierge** - Temporary Access Pass eligibility is now evaluated up-front, with clearer messaging when a pass cannot be issued (for example, when an active pass already exists).

## v2026.4.0

### 🚀 What's new

#### Enhancements

- **Credential type access indicator in presentation flows** - Presentation flow configuration in Composer now shows whether an identity has access to a requested credential type, making it easier to identify and resolve access issues before presenting credentials

### 🛠️ Notable fixes

- **Photo capture video source** - Fixed an error that prevented the camera from starting on the photo capture screen in Concierge
- **Photo capture completion** - Fixed an issue where the completion button was hidden from users after completing photo capture during issuance in Concierge
