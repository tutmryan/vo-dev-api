The January releases include infrastructure scaling and performance improvements for Redis cache and SQL database elastic pools, enhanced audit logging capabilities, new instance information features in Composer, rebranding updates for Concierge, UI improvements including simplified homepage graphs and improved chart behaviors, security vulnerability fixes, signing algorithm enhancements for JWT operations, improved application error handling, and various bug fixes and enhancements across all components.

## v2026.1.5

#### Enhancements

- Added support for more signing algorithms for OIDC provider JAR (JWT Secured Authorization Request)
- Improved error handling for application errors and updates in Composer

### 🛠️ Notable fixes

- Fixed authorization error when using OIDC provider PAR (Pushed Authorization Request)

## v2026.1.0

### 🚀 What's new

#### New features

- Implemented Microsoft Entra Access Package support for visibility of access packages associated with credentials
- Added new instance information displays with DID linked domains and hyperlinks
- Restructured the navigation menu and added feature flag filtering for a cleaner user experience

#### Enhancements

- Expanded audit logging capabilities to provide more detailed tracking of system activities
- Users assigned the Reader role can now view Remote Issuance information, with issuee contact information hidden
- Added a database performance index for faster presentation identity and wallet lookups
- Increased the display length for wallet DIDs
- Added support for user-controlled pagination length in the Composer view
- Standardized usage of a single external DID resolver site (non-ION)
- Aligned App/User onboarding documentation to use accurate display names
- Comprehensive rebranding updates for the Verified Orchestration platform

### 🛠️ Notable fixes

- Fixed an issue where the expired status filter incorrectly returned issued and cancelled records
- Updated the "OpenID Connect" icon to the correct graphic
- Improved error messages and handling during credential logo uploads
