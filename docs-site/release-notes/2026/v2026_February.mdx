## v2026.2.0

### 🚀 What's new

#### New features

- Added Apollo Studio for interactive API exploration and testing
- Implemented Composer branding customization support, allowing organizations to apply custom branding to the admin interface
- Enhanced issuance query capabilities with AND/OR filter support and configurable query limits for more flexible data retrieval

#### Enhancements

- Security improvements and dependency updates across all components

### ⚠️ Breaking Changes

- **GraphQL Schema:** Offset parameter validation updated to support zero values, enabling more consistent pagination starting from the first record (now accepts 0 and positive integers only)
