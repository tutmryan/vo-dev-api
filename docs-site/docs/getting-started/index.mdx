# Getting started

Welcome to the Verified Orchestration platform.

This guide will walk you through the following outcomes:

1. [Issue your first credential](/docs/getting-started/issue-credential)
2. [Verify your first credential](/docs/getting-started/verify-credential)

## What you'll need

Firstly, it is assumed you have an instance of Verified Orchestration and can login to the Composer.

To successfully complete this process, there are some pre-requisites that will ensure you have all the rights required to manage credentials and some resources that will be required to setup the credentials:

1. Your user has been assigned the `Credential Admin`, `Credential Issuer` roles
2. You will need logos to include in your credentials. These logos need to be in JPG or PNG format.

Finally, to accept and hold credentials, you will need a compatible digital wallet. We recommend you get started with the Microsoft Authenticator app as it is compatible with the platform and supports all features.
