# React + Apollo Client

If you are developing a React application, you can use the [Apollo Client](https://www.apollographql.com/docs/react/) to interact with the VO API.

This stack choice is best when:

- you prefer the flexibility and control of direct API access
- you prefer to use Apollo Client over a lower-level HTTP client e.g. fetch

:::tip
If you only need to issue and present credentials, you may prefer to use the [JavaScript client](/docs/guides/integration/javascript-client/) instead. It is simpler to set up and use and includes Typescript types, making it suitable for a modern React stack.
:::

Apollo documentation is comprehensive. We recommend you start with our [Apollo Client setup](./apollo-client-setup.mdx) guide, our code samples, then refer to the Apollo [React Client documentation](https://www.apollographql.com/docs/react/) as needed.
