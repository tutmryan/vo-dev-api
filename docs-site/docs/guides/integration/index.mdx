# Application integration

The Verified Orchestration platform is based on a GraphQL API which supports integration with custom applications.

We provide info and examples for the following development stacks:

- [JavaScript using client library](javascript-client)
- React + Apollo Client
- iOS Swift + Apollo Client
- Android Kotlin + Apollo Client

You may wish to use a completely different development stack. The guides and examples should still be useful as a reference. Refer to the [GraphQL API](#graphql-api) section for more info on integration to the API via whatever development stack you use.
