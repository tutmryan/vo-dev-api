This is a default page, used because the GraphQL documentation hasn't been generated yet.

If you want to see the generated GraphQL docs, run the following:

```console
npm run generate-graphql-docs
npm start
```

You may have to force a reload with `Ctrl+F5` to see the new content.

:::warning

When running the command above, it'll remove this page, i.e. `default.md`.
Make sure you don't commit the deletion of this file.

You can remove the GraphQL docs and restore the `default.md` file by running `npm run delete-graphql-docs`.

:::
