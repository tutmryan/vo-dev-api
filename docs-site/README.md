# Website

This website is built using [Docusaurus 3](https://docusaurus.io/), a modern static website generator.

### Local Development

- `npm i`
- `cp .env.template .env`
- `npm start`

It will start a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Generated content

The `api-reference` folder contains GraphQL schema documentation generated from the [GraphQL API](../README.md) project.

Schema documentation is generated by issuing the following command:

```console
npm run generate-graphql-docs
```

### Build

```
$ npm build<:env>
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

### Docker builds
The current Dockerfile can take a while to build, to speed it up you can do the following
```bash
mkdir -p /tmp/docker-cache
# From project root
docker buildx build \
  --cache-from type=local,src=/tmp/docker-cache \
  --cache-to type=local,dest=/tmp/docker-cache,mode=max \
  -t local-docs \
  -f docs-site/Dockerfile \
  .
```
